# cpp-student-record-system
CSP3341 Assignment - Student Record Management System in C++
## CSP3341 Programming Languages and Paradigms Assignment

### Overview
A C++20 console application for managing student academic records, developed as part of the CSP3341 Programming Languages and Paradigms assignment at Edith Cowan University.

### Features
- Add, view, and manage student records
- Course enrollment system
- Grade management with GPA calculation
- Console-based menu interface
- Modular object-oriented design
